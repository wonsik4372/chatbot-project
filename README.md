# RAGSystem (Chatbot Project)
    
    대학 프로젝트를 위한 RAG(Retrieval-Augmented Generation) 시스템입니다.
    
    ## 🛠 기술 스택
    * Java (JDK 17 이상)
    * Maven
    * NetBeans IDE
    * Ollama
        ㄴ nomic-embeded-model
        ㄴ gemma4:latest
        
    
    ## 🚀 주요 기능
    * 문서 데이터 임베딩 및 저장
    * 사용자 질문에 대한 관련 문맥 추출
    * 챗봇 응답 생성
