## 📚 전공 심화 과목 소개 (Course Descriptions)

본 자료는 학과에서 제공하는 주요 심화 전공 과목들의 개요 및 학습 목표를 정리한 것입니다.

---

### 💻 시스템 및 소프트웨어 개발 분야

**1. 컴퓨터 구조 (Computer Architecture)**
*   **주요 내용:** 컴퓨터 시스템의 하드웨어 구조와 동작 원리를 이해하고, 명령어 집합 구조(ISA)부터 메모리 계층 구조, 파이프라이닝, 캐시 메모리 등의 개념을 학습합니다.
*   **학습 목표:** 컴퓨터가 어떻게 작동하는지 근본적인 원리를 이해하고, 효율적인 시스템 설계를 위한 기초 지식을 습득합니다.

**2. 운영체제 (Operating System)**
*   **주요 내용:** 운영체제가 수행하는 핵심 기능(프로세스 관리, 메모리 관리, 파일 시스템, 입출력 관리 등)을 심도 있게 다룹니다. 동시성 제어, 교착 상태 해결 등 운영체제의 핵심 문제를 다룹니다.
*   **학습 목표:** 운영체제의 내부 동작 원리를 이해하고, 안정적이고 효율적인 시스템 자원 관리를 위한 프로그래밍 능력을 배양합니다.

**3. 데이터베이스 시스템 (Database System)**
*   **주요 내용:** 데이터의 구조화, 저장, 관리 원리를 학습합니다. 관계형 대수와 관계형 모델을 기반으로 데이터베이스를 설계하고, SQL을 활용한 데이터 조작 및 트랜잭션 관리 방법을 배웁니다.
*   **학습 목표:** 대규모 데이터를 체계적으로 관리하고, 데이터 무결성을 유지하는 시스템 설계 능력을 갖춥니다.

**4. 네트워크 (Network)**
*   **주요 내용:** 통신 네트워크의 기본 원리부터 OSI 7계층 모델을 기반으로 각 계층(물리, 데이터링크, 네트워크, 전송 등)의 프로토콜과 동작 방식을 학습합니다. TCP/IP 프로토콜 스택을 중심으로 심화 학습을 진행합니다.
*   **학습 목표:** 네트워크 통신 과정을 이해하고, 네트워크 문제를 분석하며, 효율적인 통신 시스템을 설계할 수 있는 능력을 기릅니다.

### 🧠 인공지능 및 데이터 분석 분야

**5. 인공지능 (Artificial Intelligence)**
*   **주요 내용:** 인공지능의 기본 개념과 역사부터 시작하여, 탐색 알고리즘, 논리 추론, 기계 학습의 기초 원리 등을 포괄적으로 다룹니다.
*   **학습 목표:** 인공지능의 다양한 접근 방식을 이해하고, 실제 문제에 적용할 수 있는 문제 해결 능력을 함양합니다.

**6. 머신러닝 (Machine Learning)**
*   **주요 내용:** 지도학습, 비지도학습, 강화학습 등 머신러닝의 주요 패러다임을 학습합니다. 회귀 분석, 분류 알고리즘(SVM, Decision Tree 등)의 원리를 이해하고 실제 데이터를 활용하여 모델을 구축하는 과정을 실습합니다.
*   **학습 목표:** 데이터를 기반으로 패턴을 학습하고 예측 모델을 개발하는 실질적인 데이터 분석 및 모델링 역량을 확보합니다.

**7. 딥러닝 (Deep Learning)**
*   **주요 내용:** 인공신경망의 구조와 원리를 심층적으로 학습합니다. CNN(이미지 처리), RNN/LSTM(자연어 처리) 등 특정 도메인에 특화된 딥러닝 아키텍처를 이해하고 구현합니다.
*   **학습 목표:** 복잡한 비정형 데이터(이미지, 텍스트 등)를 처리할 수 있는 최신 딥러닝 모델을 설계하고 구현할 수 있습니다.

### 🌐 웹 및 서비스 개발 분야

**8. 웹 프로그래밍 (Web Programming)**
*   **주요 내용:** 클라이언트(프론트엔드)와 서버(백엔드)의 구조를 이해하고, HTML, CSS, JavaScript를 활용한 동적 웹 페이지 구현 방법을 배웁니다. RESTful API 설계 및 연동을 중점적으로 다룹니다.
*   **학습 목표:** 웹 서비스의 전체 구조를 이해하고, 사용자 친화적인 인터페이스와 안정적인 서버 로직을 갖춘 웹 애플리케이션을 개발할 수 있습니다.

**9. 모바일 애플리케이션 개발 (Mobile Application Development)**
*   **주요 내용:** 안드로이드 또는 iOS 환경을 중심으로 모바일 앱의 생명주기, UI/UX 설계 원칙, 그리고 네이티브 기능(카메라, GPS 등) 연동 방법을 학습합니다.
*   **학습 목표:** 모바일 환경에 최적화된 사용자 경험을 제공하는 애플리케이션을 기획하고 개발할 수 있습니다.

### 🛠️ 공학 설계 및 방법론 분야

**10. 알고리즘 (Algorithm)**
*   **주요 내용:** 문제 해결을 위한 체계적인 절차(알고리즘)를 설계하고 분석하는 방법을 배웁니다. 시간 복잡도와 공간 복잡도를 분석하며, 정렬, 탐색, 그래프 순회 등 핵심 알고리즘을 깊이 있게 다룹니다.
*   **학습 목표:** 주어진 문제를 가장 효율적인 방법으로 해결할 수 있는 알고리즘적 사고방식과 설계 능력을 배양합니다.

**11. 소프트웨어 공학 (Software Engineering)**
*   **주요 내용:** 대규모 소프트웨어 프로젝트를 체계적으로 관리하고 개발하는 방법론을 학습합니다. 요구사항 분석, 설계, 구현, 테스트, 유지보수 등 전 개발 생명주기(SDLC) 전반에 걸친 공학적 접근을 다룹니다.
*   **학습 목표:** 기술적 역량뿐만 아니라, 프로젝트 관리 및 팀워크를 포함한 소프트웨어 개발 전반의 공학적 프로세스를 이해합니다.
