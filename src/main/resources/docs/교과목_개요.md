## [교과목 정보] 전공 교과목 상세 개요
- **소속**: 컴퓨터소프트웨어공학과
- **분류**: 전공필수 및 전공선택
- **핵심 기술**: 자바, 리눅스, SQL, 인공지능, 네트워크, 클라우드

### 1. 전공필수 과목
- **데이터구조(Data Structure)**: 스택, 큐, 리스트, 그래프, 트리 등 기본 및 고급 자료구조의 개념과 탐색, 정렬, 해슁 기법 학습
- **공학설계입문(Basic Engineering Design)**: 공학인의 기본 소양과 창의적 설계 능력을 위해 제안서 작성, 프로젝트 추진법, 팀워크 공동개발 프로젝트 수행
- **컴퓨터구조이론및실습**: 논리회로 설계, 디지털 소자, CPU, 입출력 및 메모리 장치 구조를 다루며 마이크로프로세서 설계 실습 병행

### 2. [전공선택 - 시스템 및 인프라 영역]
- **운영체제(Operating Systems)**: 프로세스, 스레드 동기화, 메모리 관리, 파일 시스템의 개념을 리눅스 사례를 통해 효율적인 자원 관리 방안 학습
- **시스템프로그래밍(System programming)**: 리눅스 환경에서 쉘 프로그래밍과 시스템 호출 인터페이스를 실습하고 시스템 소프트웨어 설계 능력 배양
- **컴퓨터네트워크(Computer Networks)**: OSI 7계층 모델과 TCP/IP 프로토콜을 이해하고 LAN, MAN, B-ISDN 등 통신 기술의 기본 모델 학습
- **클라우드시스템(Cloud System)**: 상용 클라우드 활용법과 JSP를 이용한 모바일 웹앱 구축, HTML, jQuery, JDBC 등 서버 프로그래밍 전반 학습

### 3. [전공선택 - 데이터 및 지능형 시스템 영역]
- **데이터베이스이론및실습**: 데이터 모델, DBMS 종류, SQL 질의어, 트랜잭션 처리 등 관계형 데이터베이스 설계 및 활용법 학습
- **데이터베이스응용(Database Application)**: 실무 DBMS를 활용한 데이터베이스 모델링과 전자상거래, 데이터마이닝 등 실제 업무 적용 사례 실습
- **지능정보시스템(Intelligence Information System)**: 탐색, 게임, 지식표현, 기계학습, 컴퓨터 비전 등 인공지능 전반의 이론 습득
- **인공지능시스템(Artifical Intelligence System)**: 전문가시스템, 인공신경망, 유전자 알고리즘 및 딥러닝 모델의 이해와 학습 방법을 심화 학습

### 4. [전공선택 - 소프트웨어 개발 및 응용 영역]
- **객체지향프로그래밍(I, II)**: Java, C++ 등 객체지향 언어를 이용하여 소프트웨어 개발 방법론 단계에 맞춘 응용 프로그램 작성 실습 수행
- **소프트웨어설계공학**: 폭포수 모델과 객체지향 설계 패턴을 학습하여 고품질 소프트웨어를 설계하고 개발할 수 있는 역량 함양
- **모바일컴퓨팅(Mobile Computing)**: 안드로이드 환경에서 모바일 플랫폼의 개념과 다양한 응용 프로그램 개발 방법 학습
- **캡스톤디자인(I, II)**: 외부 업체와 연계하거나 독립적인 팀 프로젝트를 통해 창의적인 주제 선정 및 실제 소프트웨어 구현·발표

### 5. [전공선택 - 이론 및 기타 영역]
- **컴퓨터알고리즘**: 알고리즘의 복잡도 분석, 효율적인 설계 기법을 다루며 주제별로 최적화된 알고리즘 처리 과정 학습
- **확률및통계(Probability and Statistics)**: 확률 이론과 랜덤 변수 분포를 기반으로 컴퓨터소프트웨어 관점의 수치적 통계 방법과 알고리즘 학습
- **컴파일러개론(Introduction to Compiler)**: 오토마타 이론을 바탕으로 프로그래밍 언어의 문법 분석, 파싱, 코드 생성 등 번역 원리 학습
